# JOBX

JOBX is a backend authentication and user management system built using Node.js.
The goal of this project is to serve as the foundation of a full job portal platform where users can create accounts, browse jobs, and apply for opportunities.

This project is currently under active development, with the backend being designed and integrated with MongoDB. The frontend and additional features will be developed in the upcoming stages.

# Features

* User signup system
* Secure password hashing using bcrypt
* MongoDB database integration
* Duplicate account detection
* Automatic user ID generation
* Environment variable management using `.env`

# Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* bcrypt
* dotenv
